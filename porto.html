Sistem Informasi Desa

Sebuah website Sistem Informasi Desa, website ini menggunakan Laravel 7 sebagai Backend dan MySQL sebagai Databasenya. Dalam website ini terdapat 2 user yaitu RT dan Warganya. RT bisa mengatur data user warganya dan kegiatan seperti Ronda, Arisan ibu-ibu, dan Arisan bapak-bapak. User yaitu warga dapat melihat informasi yang seperti kegiatan Ronda, Arisan ibu-ibu, dan Arisan bapak-bapak.

Sistem Informasi Perpustakaan

Sebuah website Sistem Informasi Perpustakaan, website ini menggunakan Laravel 7 sebagai Backend dan MySQL sebagai Databasenya. Dalam website ini terdapat 1 user yaitu hanya admin. Admin bisa mengatur alur peminjaman-pengembalian buku, data siswa, data buku, dan export data ke export/PDF.
